# Orvynd

Orvynd is being built.

A simple public landing page for Orvynd.

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

This project is deployed to GitHub Pages using GitHub Actions.

The deployment workflow is located at:

```text
.github/workflows/deploy.yml
```

The custom domain is:

```text
orvynd.com
```

## Notes

Orvynd is currently in its foundation phase.
More will appear as the foundation takes shape.
